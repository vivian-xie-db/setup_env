# Databricks CLI Commands Reference (llm.txt)

> Comprehensive reference for Databricks CLI commands (v0.205+)
> Base documentation: https://docs.databricks.com/aws/en/dev-tools/cli/commands

## Command Structure

```bash
databricks <command-group> <command-name> [subcommand] [flags]
databricks <command-group> -h                    # Show help for command group
databricks <command-group> <command> -h          # Show help for specific command
```

## Global Flags

Available to all commands:
- `--debug` - Enable debug logging
- `-h, --help` - Display help
- `--log-file string` - File to write output logs to
- `--log-format {text|json}` - Log format type (default: text)
- `--log-level string` - Log format level
- `-o, --output {text|json}` - Command output type (default: text)
- `-p, --profile string` - Profile name from ~/.databrickscfg (default: DEFAULT)
- `--progress-format {default|append|inplace|json}` - Progress display format
- `-t, --target string` - Bundle target to use

## Quick Reference Examples

```bash
# List all clusters
databricks clusters list

# Create a secret scope
databricks secrets create-scope <scope-name>

# Import a notebook
databricks workspace import <local-path> <workspace-path>

# List Unity Catalog tables
databricks tables list --catalog <catalog> --schema <schema>

# Deploy a Databricks app
databricks apps deploy <app-name>

# Get job run status
databricks jobs get-run --run-id <run-id>

# Query a serving endpoint
databricks serving-endpoints query <endpoint-name>

# List vector search endpoints
databricks vector-search-endpoints list
```
