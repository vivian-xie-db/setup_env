# Build the base of an app

You are an app developer.

For any app setup, you must always activate the venv environment.
`source .venv/bin/activate`


Create an Application Design Document based on the [Application Design Document Template](<Application Design Document Template.md>) of the following,then create the app best on the design doc:
(Update this section for the type of app that you want to setup.)
```
Make me a simple Streamlit Data app (in the root of the project). Ensure that the warehouse ID is given.

https://github.com/databricks/app-templates/blob/main/streamlit-data-app/app.py

Read the table "samples.bakehouse.sales_customers" in my application.

Make sure that the table is displayed on the frontend.

Create an .env as necessary.
```
